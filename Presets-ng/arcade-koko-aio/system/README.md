# Additional installation requirement for Game & Watch Handheld Artwork and MAME-specific overlays

Game & Watch handhelds with dual screen only work in RetroArch if you also install the respective files contained in this directory in RetroArch/system/mame/artwork - as shown hin this folder structure. Example: Game & Watch Donkey Kong

The same applies to overlays in MAME that colorize the pixels, such as in Space Invaders Deluxe. The necessary files are also placed here.

This is additional installation, koko-aio specific artwork is still required.
